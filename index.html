Drawing PDF Splitter

Upload a searchable combined PDF, define the scan rectangle, preview the scanned area, then split all pages into individual PDFs and download a ZIP with the outputs and register CSV.

Runs entirely in the browser. Input PDFs stay on the user's machine unless they choose to upload them elsewhere.

Select a searchable PDF to start.

Detection Pattern

Examples: \d = any digit, [A-Z] = any capital letter, \. = actual dot.

If you need a real special character like . or (, add \ before it.

When enabled, common OCR mistakes are corrected where the pattern expects them, such as O→0, I/l→1, and 0→O.
